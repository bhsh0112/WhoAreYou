# 辈分计算器

一个美观的 Windows 桌面应用，用于计算复杂的亲戚关系并得出正确的称呼。

## 功能特点

- 🎨 **美观的界面设计** - 现代化的计算器风格界面
- 🧮 **智能关系计算** - 支持复杂的关系链计算（如"妻子的父的子"）
- 📝 **历史记录** - 自动保存计算历史，方便查看和复用
- 💾 **离线运行** - 完全离线，无需网络连接
- ⚡ **快速响应** - 流畅的用户体验

## 技术栈

- **Electron** - 跨平台桌面应用框架
- **React** - UI 框架
- **TypeScript** - 类型安全
- **Tailwind CSS** - 现代化样式
- **Vite** - 快速构建工具

## 开发环境要求

- Node.js >= 18.0.0
- npm 或 yarn

## 安装依赖

```bash
npm install
```

## 开发模式

启动开发服务器（同时启动 Vite 和 Electron）：

```bash
npm run electron:dev
```

或者分别启动：

```bash
# 终端1: 启动 Vite 开发服务器
npm run dev

# 终端2: 启动 Electron（等待 Vite 启动后）
npm run electron:dev
```

## 构建应用

### 构建前端和 Electron 代码

```bash
npm run build
```

### 打包为 Windows 安装包

```bash
npm run build:app
```

打包后的安装包会在 `release` 目录中。

## 使用方法

1. 点击关系按钮输入关系链（如：父、母、夫、妻、兄、弟、姐、妹、子、女）
2. 使用"的"按钮连接关系
3. 点击"="按钮计算结果
4. 点击"历史"按钮查看历史记录
5. 点击"清空(AC)"清除输入
6. 点击"删除(DEL)"删除最后一个字符

## 示例

- `妻子的父` → 岳父
- `母的母` → 外婆
- `兄的妻` → 嫂子
- `子的子` → 孙子

## 注意事项

- 各地称呼可能不同，计算结果仅供参考
- 历史记录最多保存 50 条
- 历史记录存储在本地，不会上传到服务器
